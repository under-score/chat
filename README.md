# projector.chat

let your audience chat live  
works well with https://github.com/under-score/projector  
does threading, uses initals, allow preview and pictures  

<img src="screen1.jpg" width="70%" height="70%">  

seven easy steps for setup:
1. move this project to a subdirectory of the projector app
2. create a cache subdirectory if you are allowing picture attachments
3. standard set up is admin preview before "publication" - this can be changed in client.php
4. point your admin browser to http://x.x.x.x/projector/projector.chat/admin.php Called for the first time a database will be created for you 
5. point your display browser to http://x.x.x.x/projector/projector.chat/chat.php 
6. let the users enter their messages at http://x.x.x.x/projecor/projector.vote/client.php  
7. whenever there is a new message in admin.php, change it from 0 to 1, by clicking on the second entry of the row, and it will appear in less athn 1 second on the screen
